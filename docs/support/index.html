TeleDrive Support

Need help with TeleDrive? This page explains how to contact support and solve common problems.

Contact Support

For questions, bug reports, or account-related problems, contact us by email.

your-email@example.com

Login Problems

TeleDrive uses Telegram authentication. If you cannot log in, check these items:

  • Make sure your phone number is correct.
  • Check the verification code sent by Telegram.
  • If your Telegram account uses 2FA, enter your 2FA password.
  • Make sure your internet connection is working.

Upload and Download Issues

If upload or download does not work correctly, try these steps:

  • Keep the app open while the file operation is running.
  • Check your internet connection.
  • Try with a smaller file to test the connection.
  • Make sure your device has enough free storage.

Folders and Channels

TeleDrive uses Telegram Saved Messages and private channels as storage locations.

  • Saved Messages is used as the default storage location.
  • Private channels can be used as folders.
  • You may need admin permission in a channel to upload files there.

Supported Platform

TeleDrive currently focuses on Android because the native Telegram TDLib integration is implemented for Android.

iOS support is not currently available.
