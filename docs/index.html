Built for personal Telegram cloud storage

TeleDrive connects to Telegram through TDLib and helps you manage files from Saved Messages and private channels with a clean mobile interface.

1

Telegram-based storage

Use Saved Messages as the default storage location and private Telegram channels as separate folders.

2

Upload and download files

Upload documents and media to selected folders, then download them back to your device with progress tracking.

3

Automatic file categories

Organize files into useful groups such as documents, images, videos, audio, PDFs, and archives.

Secure Telegram authentication

TeleDrive supports Telegram login with phone number, verification code, and two-factor authentication. The app stores sessions securely on the device, so users do not need to log in every time.
